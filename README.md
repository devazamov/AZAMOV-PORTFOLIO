# AZAMOV OFFICIAL — Portfolio

A highly interactive personal portfolio for **Zarifjon A'zamov** (AZAMOV) — IT Blogger and Developer from Jizzax, Uzbekistan.

## Features

- ⚡ Built with Next.js 15 & React
- 🎨 Cyberpunk / Glassmorphism design
- 🌐 Multilingual support (English & Uzbek)
- 🎭 Dark/Light theme switcher
- 🖱️ Custom cursor with physics interactions
- 🎬 Smooth scroll with Lenis
- ✨ Framer Motion animations
- 📱 Fully responsive

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
